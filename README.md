Strategy Skills — Consulting-Grade Thinking for AI Agents

Most AI agents produce strategy advice that reads like a Wikipedia summary with bullet points. These skills fix that by teaching agents the same scoping and analytical methodology used by top-tier management consulting firms (MBB) to structure engagements.

Three skills. Each one addresses a specific failure mode in AI-generated strategy work.

Decision Framing solves the most common problem: vague questions produce vague answers. This skill trains agents to reject fuzzy problem statements ("what's our AI strategy?") and reframe them as specific decisions with named options ("should we build, acquire, or partner?"). It extracts a decision statement, identifies 2-3 live alternatives, defines the strategic objective, and derives a governing question that anchors the entire analysis. No analysis begins until the decision is sharp.

Constraint-First Reasoning changes how agents evaluate options. The default AI behavior is to weigh pros and cons — which produces balanced-sounding analysis that doesn't actually help anyone decide. This skill teaches agents to identify the 1-2 binding constraints that limit the decision space, then test every option against those constraints before evaluating anything. Constraints are gates that eliminate options, not factors to weigh. An option that violates a binding constraint is dead regardless of its other merits. This mirrors how experienced consultants and operators actually think.

Audience-Calibrated Communication ensures the output is framed for the person who needs to act on it. A skeptical CFO needs risk mitigation and unit economics up front. A board that's already aligned needs execution specificity and progress metrics. An investment committee needs thesis validation and return scenarios. This skill trains agents to identify the audience, understand what they already believe, and adjust structure, evidence selection, and tone accordingly.
